## Installation

(Docker)

```bash
docker run -ti audy/dotfiles
```

(OSX or Ubuntu)

```bash
curl -L https://git.io/vu9Do > install.sh
chmod +x install.sh
# read install.sh carefully
./install.sh
```

For macOS-specific settings:

```bash
./macos.sh # requires reboot
```
